# プライバシーポリシー

最終更新日: 2026年2月15日

[アプリ名]（以下「本アプリ」）の運営者（以下「運営者」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシー（以下「本ポリシー」）は、本アプリにおける個人情報の取り扱いについて定めるものです。

---

## 1. 収集する情報

### 1.1 ユーザーが提供する情報

| 情報 | 取得タイミング | 必須/任意 |
|------|-------------|----------|
| メールアドレス | アカウント登録時 | 必須 |
| パスワード（ハッシュ化） | アカウント登録時 | 必須 |
| 表示名 | アカウント登録時 | 必須 |
| Googleアカウント情報 | Googleログイン時 | 任意 |
| Apple ID情報 | Appleログイン時 | 任意 |
| レビュー・感想テキスト | レビュー作成時 | 任意 |
| メモ | メモ作成時 | 任意 |
| タグ | タグ作成時 | 任意 |

### 1.2 自動的に収集する情報

| 情報 | 目的 |
|------|------|
| デバイス情報（OS、バージョン） | サービス最適化・不具合対応 |
| FCMデバイストークン | プッシュ通知の送信 |
| ブックマーク登録した小説URL | 更新チェック機能の提供 |
| アプリ利用日時 | サービス改善 |

### 1.3 収集しない情報

- 位置情報
- 連絡先情報
- 通話履歴
- 閲覧中の小説本文
- 対象サイトのアカウント情報（ID・パスワード等）

---

## 2. 情報の利用目的

収集した情報は、以下の目的でのみ利用します。

1. **本サービスの提供・運営**
   - アカウント管理、認証処理
   - ブックマーク管理、更新通知の送信
   - レビュー・感想・メモの保存と表示

2. **サービスの改善**
   - 利用状況の分析によるUX改善
   - 不具合の検知と修正

3. **ユーザーへの連絡**
   - サービスに関する重要なお知らせ
   - 利用規約・プライバシーポリシーの変更通知

4. **広告の表示**
   - Amazonアフィリエイトプログラムによる関連書籍の表示
   - ※行動ターゲティング広告は行いません

## 3. 情報の保存場所

| 情報種別 | 保存場所 | 暗号化 |
|---------|---------|--------|
| アカウント情報 | Supabase (AWSインフラ) | TLS通信 + 保存時暗号化 |
| ブックマーク・レビュー等 | Supabase (AWSインフラ) | TLS通信 + 保存時暗号化 |
| FCMトークン | Supabase (AWSインフラ) | TLS通信 + 保存時暗号化 |
| PDFファイル | ユーザーの端末のみ | 端末のセキュリティに依存 |

## 4. 第三者サービスとの情報共有

本アプリは以下の第三者サービスを利用しており、それぞれのプライバシーポリシーに従い情報が取り扱われます。

### 4.1 Supabase（バックエンド基盤）

- **共有する情報**: アカウント情報、ブックマーク・レビュー等のユーザーデータ
- **目的**: データベース管理、認証、APIサーバー
- **プライバシーポリシー**: https://supabase.com/privacy

### 4.2 Firebase / Google（プッシュ通知）

- **共有する情報**: FCMデバイストークン、通知メタデータ
- **目的**: プッシュ通知の配信
- **プライバシーポリシー**: https://firebase.google.com/support/privacy

### 4.3 Google（ソーシャルログイン）

- **共有する情報**: OAuth認証に必要な情報（メールアドレス、表示名）
- **目的**: Googleアカウントによるログイン
- **プライバシーポリシー**: https://policies.google.com/privacy

### 4.4 Apple（ソーシャルログイン）

- **共有する情報**: Apple IDによる認証情報
- **目的**: Apple IDによるログイン
- **プライバシーポリシー**: https://www.apple.com/legal/privacy/

### 4.5 Amazon（アフィリエイト広告）

- **共有する情報**: なし（広告表示はサーバー側で処理）
- **目的**: 関連書籍のアフィリエイト広告表示
- **プライバシーポリシー**: https://www.amazon.co.jp/gp/help/customer/display.html?nodeId=GX7NJQ4ZB8MHFRNJ

### 4.6 情報を共有しない第三者

上記以外の第三者に対して、ユーザーの個人情報を販売、貸与、または共有することはありません。ただし、以下の場合を除きます。

- 法令に基づく開示要請があった場合
- ユーザーの生命、身体または財産の保護のために必要な場合
- ユーザーの明示的な同意がある場合

## 5. データの保持期間

| データ | 保持期間 |
|--------|---------|
| アカウント情報 | アカウント削除から30日後に完全削除 |
| ブックマーク・レビュー等 | アカウント削除から30日後に完全削除 |
| 通知履歴 | 作成から90日後に自動削除 |
| クロールログ | 作成から30日後に自動削除 |
| FCMトークン | ログアウト時に即時削除 |

## 6. ユーザーの権利

ユーザーは以下の権利を有します。

### 6.1 アクセス権

- 自身のデータにアクセスし、内容を確認する権利があります。
- アプリの設定画面からデータのエクスポート（JSON形式）が可能です。

### 6.2 訂正権

- 自身のプロフィール情報、レビュー、メモ等をいつでも編集・訂正できます。

### 6.3 削除権

- 個別のブックマーク、レビュー、メモ等を自由に削除できます。
- アカウント削除を申請し、全てのユーザーデータの削除を求めることができます。
- アカウント削除はアプリの設定画面から、またはメールでの申請により実行できます。

### 6.4 データポータビリティ権

- 設定画面の「データエクスポート」機能から、ブックマーク・レビュー・タグ等のデータをJSON形式でダウンロードできます。
- ローカル保存のPDFファイルはエクスポートに含まれます。

### 6.5 通知のオプトアウト

- プッシュ通知はアプリの設定画面からいつでも無効化できます。
- 通知の種別ごと（新エピソード / 新作 / 完結）に個別にON/OFF設定が可能です。
- 端末のOS設定からも通知を無効化できます。

## 7. 児童のプライバシー

本サービスは13歳未満の児童を対象としていません。13歳未満であることが判明した場合、該当するアカウントおよびデータを速やかに削除します。

---

## 8. セキュリティ対策

ユーザーの情報を保護するため、以下のセキュリティ対策を実施しています。

| 対策 | 内容 |
|------|------|
| 通信暗号化 | 全通信をTLS (HTTPS) で暗号化 |
| パスワード保護 | bcryptによるハッシュ化（Supabase Auth標準） |
| アクセス制御 | Row Level Security (RLS) による行レベルのアクセス制御 |
| 認証トークン | JWT (JSON Web Token) による安全な認証 |
| APIキー管理 | サーバーサイドの環境変数で管理、クライアントに非公開 |

---

## 9. Cookieおよびトラッキング

- 本アプリはWebアプリではないため、Cookieは使用しません。
- 行動トラッキングや広告IDの取得は行いません。
- サードパーティのトラッキングSDKは組み込みません。
